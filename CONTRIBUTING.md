# Contributing

Thanks for helping improve MightBeHuman.

## Setup

```bash
npm install
npm test
npm run typecheck
```

## Workflow

- Keep changes small and focused.
- Add or update tests for behavior changes.
- Run the workspace checks before opening a pull request.
- Prefer deterministic, local-only changes.

## Pull requests

- Describe the problem and the fix.
- Include verification steps.
- Mention any tradeoffs or follow-up work.
