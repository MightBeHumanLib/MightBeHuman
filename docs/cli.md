# CLI

The CLI is the fastest way to exercise the engine.

## Commands

- `analyze`
- `humanize`
- `compare`
- `detect`
- `score`
- `preserve`
- `export`
- `benchmark`
- `pipeline-debug`

## Examples

- `node apps/cli/src/main.ts analyze input.txt`
- `node apps/cli/src/main.ts humanize input.txt --out output.txt`
- `node apps/cli/src/main.ts compare source.txt target.txt --json`
