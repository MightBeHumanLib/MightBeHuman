# Brand Guide

## Logo

Use the provided logo on dark backgrounds with generous spacing.

- Primary asset: `brand/logo.png`
- Web assets: `apps/web/public/logo.png`
- Docs assets: `apps/docs/public/logo.png`

## Voice

- Clear
- Technical
- Minimal
- Direct

## Color palette

- Background: near-black
- Surface: charcoal
- Foreground: white
- Accent: soft white / gray

## Usage notes

- Prefer dark, low-noise layouts.
- Keep the logo unobstructed.
- Use short labels and clean typography.
